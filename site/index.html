FAQ

        

          
Can I self-host everything?

          
Yes. The open-source core runs locally and can be deployed anywhere. Production deliverability is why the cloud exists.

        

        

          
Is this just another email MCP server?

          
No. Nerve is an agent runtime: threads, memory, triage, extraction, policy drafting, so you do not build glue.

        

        

          
Do you support custom domains?

          
Yes on Cloud (and self-host). Local dev also works via tunneling.

        

        

          
What models do you use for triage/extraction?

          
Bring-your-own-model is supported in the self-hosted engine. Cloud offers managed defaults and tuning.

        

        

          
How do you prevent abuse/spam?

          
Outbound is gated, reputation is segmented, and policies + allowlists are default.
